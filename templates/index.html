Emergency Power Response Dashboard

How to Use This Dashboard

Switching between nowcasting and forecasting

Select one of the buttons in the top right corner
• Live: Current weather data
• Predicted: Average weather data today
• +1d, +2d, ..., +6d: Average weather data over the next week

Viewing Outage Predictions

Use the panel on the left to see the current outage severity predictions.
Colors indicate severity:
• Yellow (minor)
• Orange (moderate)
• Red (severe).
Click on a county to zoom into it on the map and display the detailed prediction and feature importance panel on the right.

Viewing Detailed Predictions and Feature Importance

On the feature importance panel you can view the predicted probability for a given county. You can also view the predicted scope and duration for that county.

Below the weather inputs is the model analysis and feature importance. Select one of the models to view the feature importance for that model.
• High positive values indicate a strong positive impact for the model (e.g. +0.93).
• High negative values indicate a strong negative impact (e.g. -1.52).
• Values close to 0 have little impact.

Interacting with the Map

Click on a county to zoom into it on the map and display the feature importance panel on the right.
Hover over a county to see the scope and duration predictions.
